Proyecto de Base de Datos Desarrollo e Implementación de una Aplicación Web para el análisis de sentimiento de Tweets en la zona del Ecuador, respecto al índice de aprobación del presidente de la Republica del Ecuador Lenin Moreno Garcés

Objetivo

Identificar los índices de aprobación de la presidencia de Lenin Moreno, a través de un análisis de sentimiento el cual estará basado en la recolección de información de los tweets realizados dentro del territorio ecuatoriano mediante la geolocalización y el uso de la base de datos Mongo como gestor.
